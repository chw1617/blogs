# 文件查找

一 模块规范

  • nodejs 对common.js进行了支持与实现
  • 在node中，每个文件都是一个模块
  • 每个模块中包含common.js规范的核心变量：require、module.exports、exports(是module.exports的一个引用)
  • 模块化的核心思想是导入与导出，在node中通过module.exports,exports负责对模块的内容进行导出，通过require导入其他模块（原生模块，自定义模块，第三方模块）

二 查找策略

  1. 先是从缓存区的找
  2. 原生模块 path|fs|http，加载完成之后又进行缓存
  3. 绝对路径 / 和相对路径 ./ 自定义的第三方模块，加载完成之后进行缓存
  4. 第三方模块 node-module ,这里先从当前编写的文件开始查找，没有找到，依次递归的往上层目录查找，直到根目录
